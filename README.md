

#  Cardápio Digital

Projeto de um cardápio digital moderno, responsivo e interativo desenvolvido com React e TypeScript.
A aplicação permite visualizar produtos, adicionar itens ao carrinho, alterar quantidades e simular uma experiência completa de pedidos online.

---

## Tecnologias Utilizadas

* React
* TypeScript
* Vite
* Tailwind CSS
* Context API
* React Hooks
* CSS Responsivo

---

## Funcionalidades

✅ Listagem de produtos
✅ Interface moderna e responsiva
✅ Carrinho de compras funcional
✅ Adição e remoção de produtos
✅ Atualização de quantidade dos itens
✅ Cálculo automático do valor total
✅ Estrutura organizada em componentes
✅ Gerenciamento global de estado com Context API

---

---

## Funcionamento do Carrinho

O projeto utiliza Context API para compartilhar os dados do carrinho entre os componentes da aplicação.

### Principais funcionalidades:

* Adicionar produtos ao carrinho
* Remover produtos
* Atualizar quantidade
* Abrir e fechar carrinho
* Calcular total de itens
* Calcular valor total da compra
